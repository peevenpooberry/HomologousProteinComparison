# Protein Comparison Dashboard and Tools

## Inputs:
The user will be able to enter `.cif`, `.pdb`, or PDB ID's. In the case of PDB IDs being given, the program will initiate an API request to PDB to get the relevant protein structure files.

## Features:
To visualize the tools I am currently looking into python runable scripts similar to PyMol or ChimeraX that will allow interactive visualization of protein structure.

I also plan to add many quantitate tools using measures very common within the field. Like a Residue vs pLDDT plot for selected proteins, and a heatmap to visualize RSMD clustering of high confidence alpha carbons of user selected protein structures.
